JavaScript

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

Introdução Operadores de Comparação Javascript

Quando estudamos operações matemáticas em JS, aprendemos sobre operadores de atribuição, porque sempre que os usamos, estamos fazendo atribuições. Por exemplo: var age = 21; O operador igual atribui o valor 21 à variável idade. É como se você tivesse feito um pedido de JavaScript.

Agora, vamos fazer uma pergunta sobre o JavaScript em vez de fazer um pedido. No entanto, tem uma característica: só pode responder True ou False a perguntas

Para comparar dois valores usamos os operadores de comparação (comparison operators). Veja abaixo uma tabela que mostra os operadores de comparação JavaScript.

Tabela comparativa do JavaScript

Operador Significado
< Menor que
> Maior que
<= Menor ou igual que
>= Maior ou igual que
== Igual ao
!= Diferente de
