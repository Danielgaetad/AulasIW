DESENVOLVIMENTO PARA INTERNET- HTML E CSS

O que é HTML?

HTML- abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação usada para construir páginas da web e documentos HTML podem ser interpretados por navegadores. Essa tecnologia é o resultado da combinação entre os padrões HyTime e SGML.

Hipertexto- é o termo que remete a um texto ao qual se agregam outros conjuntos de informação na forma de blocos de textos, palavras, imagens ou sons, cujo acesso se dá através de referências específicas, no meio digital denominadas hiperligações.

O que é W3C?

O World Wide Web Consortium é a principal organização de padronização da World Wide Web. É composto por uma aliança internacional com 450 membros, reunindo empresas, agências governamentais e organizações independentes, com o objetivo de estabelecer padrões para a criação e interpretação de conteúdo da Web .

O que é CSS?

CSS é a sigla para Cascading Style Sheets, ou seja, Folhas de Estilo em Cascatas. É uma maneira de dar estilo ao código criado por linguagens como HTML, XML ou XHTML, por exemplo. De forma prática, ela funciona como uma camada de personalização ao conteúdo visível.

A diferença entre .HTML e .HTM

A extensão .htm é característica do MS-DOS depois Windows. Para quem nunca trabalhou com DOS não deve saber, mas ele não aceitava mais que três caracteres na extensão de qualquer arquivo, daí vem a origem das extensões com três letras, por exemplo: .doc, .exe, .txt, etc.

A extensão .html é característica do UNIX que suportava mais de três caracteres.

Para evitar problemas, foi feita uma conexão e, para fins de compatibilidade, duas extensões foram mantidas. Portanto, é correto dizer que não há diferença entre htm e html, ou seja, o conteúdo da página é o mesmo. Se houver dois documentos com o mesmo nome no mesmo site, um denominado index.html e o outro denominado index.htm, o servidor sempre exibirá index.htm
